# Party Palace

Event decoration and rental services website with integrated booking system.

## Features
- Product catalog (arches, columns, centerpieces, walls, services)
- Gallery with product images
- Contact form
- Waiver and contract signing
- Booking system
- Supabase backend integration

## Tech Stack
- HTML5, CSS3, JavaScript
- Supabase (PostgreSQL database)
- Static site hosting

## Deployment
This site is deployed on Railway.

## Local Development
1. Open `index.html` in your browser
2. Or use a local server: `npx serve .`
